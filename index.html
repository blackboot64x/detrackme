Tails OS (Amnesic Incognito Live System)

Tails is a security-focused, portable operating system designed to protect privacy and anonymity.

Purpose:
Runs from a USB stick or DVD without leaving traces on the device
Routes internet traffic through the Tor network
Includes pre-configured apps for secure communication (email, messaging, file encryption)

Note:
Tails is not intended for daily use
It's designed for journalists, activists, or anyone needing strong anonymity
No persistent storage unless manually enabled
